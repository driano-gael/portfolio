Présentation de l'Application

L'objectif principal de cette application est d'automatiser la communication avec une API fournissant des statistiques footballistiques, dans le but de créer un tableau de bord offrant une consultation complète des données relatives au football à l'échelle mondiale.
Dans le cadre d'un projet visant à fournir un outil centralisé pour l'analyse des performances et des tendances du football à travers le monde, ce composant spécifique est chargé de collecter les informations fournies par l'API "API-FOOTBAL". Il assure également le nettoyage des données, garantissant ainsi leur qualité et leur fiabilité avant de les intégrer dans une base de données MySQL.

Contexte de développement

Cette application a été conçue dans le cadre d'un projet scolaire réalisé pour le BTS SIO. Elle a été spécifiquement créée pour répondre à une demande émanant d'un acteur passionné du milieu du football, qui avait un besoin urgent d'accéder à une vaste gamme de données statistiques.
L'application a été déployée sur un serveur d'hébergement afin de la rendre accessible à tout moment.
